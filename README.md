### 📖 FARIZAL COFFE Website
Deskripsi Proyek

FARIZAL COFFE adalah proyek website untuk sebuah coffee shop yang menghadirkan pengalaman kopi premium bagi pelanggan. Website ini menampilkan menu, informasi tentang kopi dan barista, serta fitur pemesanan online sederhana.

Proyek ini dibuat menggunakan HTML, CSS, dan JavaScript, dan dapat dijalankan langsung di Replit.

Fitur Utama
Homepage dengan hero section, cerita brand, menu unggulan, dan testimoni pelanggan
Menu Page dengan daftar kopi, harga, dan kategori (Hot, Cold, Signature)
Order Page dengan keranjang belanja dan tombol checkout
Contact Page dengan formulir kontak, peta, dan informasi WhatsApp/email
Responsive Design untuk tampilan optimal di desktop, tablet, dan mobile
Animasi dan efek hover untuk pengalaman pengguna yang interaktif
Cara Menjalankan
Buka proyek ini di Replit
Klik tombol Run untuk menjalankan server
Akses website melalui URL yang disediakan oleh Replit
